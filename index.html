Itô process

finance/ito.py

The foundational building block. A continuous-time stochastic process driven by Brownian motion. The quadratic variation converges to $T$ as $dt \to 0$ — the core result of Itô calculus.

SDE
$dX_t = b \, dt + \sigma \, dW_t$
bdrift rate sigmadiffusion coefficient dWBrownian increment $\sim \mathcal{N}(0, dt)$ 
import numpy as np
from finance import ito

X  = ito.simulate(b=0.05, sigma=0.2)   # (n_paths, n_steps)
qv = ito.quadratic_variation(dW)        # converges to T = 1.0

Geometric Brownian motion

finance/gbm.py

The Black-Scholes model for equity prices. Solved exactly via the log-Euler scheme — no discretisation error in the price level, only in the drift and diffusion parameterisation.

SDE
$dS_t = \mu S_t \, dt + \sigma S_t \, dW_t$
Exact solution (Itô's lemma)
$S_t = S_0 \exp\!\left[\left(\mu - \tfrac{1}{2}\sigma^2\right)t + \sigma W_t\right]$
S0initial price mulog-return drift sigmavolatility 
from finance import gbm

S = gbm.simulate(S0=100, mu=0.05, sigma=0.2)  # (n_paths, n_steps+1)
r = gbm.log_returns(S)                         # (n_paths, n_steps)

CIR model

finance/cir.py

Cox-Ingersoll-Ross short-rate model. Mean-reverts to $\alpha/\beta$ and keeps rates non-negative when the Feller condition holds. Used to model interest rates and stochastic volatility.

SDE
$dr_t = (\alpha - \beta r_t)\, dt + \gamma \sqrt{r_t}\, dW_t$
Feller condition (rates stay positive)
$2\alpha > \gamma^2$
alphaspeed of mean reversion × long-run mean betaspeed of mean reversion gammavolatility of volatility r0initial short rate 
from finance import cir

r = cir.simulate(r0=0.05, alpha=0.3, beta=2.0, gamma=0.1)
# prints: Feller condition (2α > γ²): 0.6000 > 0.0100  →  True

Heston model

finance/heston.py

Stochastic volatility model that fixes GBM's constant-vol assumption. The price and variance processes are correlated through $\rho$, which produces the observed volatility smile. The variance process is itself a CIR process.

Price SDE
$dS_t = (r - q)\, S_t\, dt + \sqrt{V_t}\, S_t\, dW_1$
Variance SDE
$dV_t = \kappa(\bar{v} - V_t)\, dt + \eta\sqrt{V_t}\!\left(\rho\, dW_1 + \sqrt{1-\rho^2}\, dW_2\right)$
kappamean-reversion speed of variance mlong-run variance $\bar{v}$ etavol-of-vol rhoprice-vol correlation (typically negative) 
from finance import heston

S, V = heston.simulate(
    S0=100, V0=0.04, r=0.03, q=0.0,
    kappa=2.0, m=0.04, eta=0.3, rho=-0.7
)
# S and V both shape (n_paths, n_steps+1)

Lévy jump-diffusion

finance/levy.py

Extends GBM with a compound Poisson jump component to capture sudden large moves (crashes, earnings surprises). Jumps arrive at rate $\lambda$ and are log-normally distributed.

Log-process
$X_t = \mu t + \sigma W_t + \sum_{i=1}^{N_t} J_i, \quad J_i \sim \mathcal{N}(\mu_J, \sigma_J^2)$
Price process
$S_t = S_0 \exp(X_t)$
lambda_jumpaverage jumps per year jump_meanmean jump size $\mu_J$ jump_stdjump volatility $\sigma_J$ 
from finance import levy

St = levy.simulate_price(
    S0=100, mu=0.10, sigma=0.15,
    lambda_jump=10, jump_mean=0.0, jump_std=0.03
)

Value at Risk

finance/var.py

Three VaR methods: historical simulation, rolling window, and parametric (Gaussian). Rolling VaR is a sliding-window algorithm — exactly the same pattern as NeetCode's max-sum subarray, applied to risk.

Historical VaR
$\text{VaR}_\alpha = -Q_{\,1-\alpha}(r_1, \ldots, r_T)$
Parametric (Gaussian)
$\text{VaR}_\alpha = -(\mu - z_\alpha \sigma), \quad z_\alpha = \Phi^{-1}(\alpha)$ 
from finance import var
import numpy as np

returns    = np.random.normal(-0.001, 0.015, 252)
daily_var  = var.historical_var(returns, confidence=0.95)
roll_var   = var.rolling_var(returns, window=20)
param_var  = var.parametric_var(mu=-0.001, sigma=0.015)

Mean-variance hedging

finance/mv_hedge.py

Variance-optimal hedging of a European call on a discrete return lattice. Uses the variance-optimal martingale measure (VOMM) — an Esscher transform of the physical measure that minimises the expected squared hedging error.

VOMM risk-neutral measure
$Q(R_i) = P(R_i) \cdot \frac{1 - a(R_i - R_f)}{b}$
Optimal hedge (delta)
$\xi_t^* = \frac{\mathbb{E}^P[(V_{t+1} - R_f V_t)(R - R_f)]}{\mathbb{E}^P[(R - R_f)^2]}$
Tidxnumber of rebalancing dates Rsafegross risk-free return per period lnRlattice log-returns PDistrphysical probabilities 
from finance import mv_hedge
import numpy as np

lnR    = np.array([0.06, 0.04, 0.02, 0.00, -0.02, -0.04, -0.06])
P      = np.array([0.013, 0.067, 0.273, 0.384, 0.199, 0.050, 0.014])
Rsafe  = 1.04 ** (1/52)

eps2, Q, price = mv_hedge.hedge(S0=100, strike=100, Tidx=7, Rsafe=Rsafe, lnR=lnR, PDistr=P)
print(f"VOMM call price: {price:.4f}")
